# DotNetBrushUp

A C# console **scratchpad** of practice exercises kept around for brushing up on .NET fundamentals — collections, LINQ, dependency-graph problems, common interview-style algorithms, etc.

Most exercises are commented-out blocks in `Program.cs`; uncomment whichever one you want to run, or copy them into [LINQPad](https://www.linqpad.net/).

## Run

```bash
dotnet run --project DotNetBrushUp
```

## Stack

C# / .NET console app.
